# The `files` module.

The `files` module contains various commands for working with files in nimbsh.

## `rf`

The `rf` command reads the entirety of a file and outputs it to the console.

```
> rf names.txt

srsxnsh
terry
````



## `fw`

The `fw` command writes the input to the specified file.

```
> fw lol file.txt

# file.txt
lol

```
