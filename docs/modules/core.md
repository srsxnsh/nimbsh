# The Nimbsh `core` module.

The `core` module contains the most basic and essential commands in the nimbsh shell.

## `echo`

The `echo` command is used to output text to the console.

```
> echo Hello, nimbsh.

Hello, nimbsh.
```

## `go`

The `go` command can be used to change the current working directory.

```
> go new/path
```

## `l`

The `l` command can be used to list all files and directories in the current directory,
or a specified other directory.

```
> l

hi.txt
people/
places/

> l people

srsxnsh.yaml
terry.yaml
```

## `exit`

The `exit` command is used to exit the shell.

```
> exit

toodles.
```

## `m`

The `m` command can be used to make a new file.

```
> m newfile.txt
```

## `md`

The `md` command can be used to create a new directory.

```
> md mydir
```

## `r`

The `r` command is used to remove files.

```
> r deleteme.txt
```

## `rd`

The `rd` command is used to remove directories.

```
> rd directory_name
```
