# The Nimbsh `core` module.

The `core` module contains the most basic and essential commands in the nimbsh shell.

## `echo`

The `echo` command is used to output text to the console.

```
> echo Hello, nimbsh.

Hello, nimbsh.
```

## `go`

The `go` command can be used to change the current working directory.

```
> go new/path
```

## `exit`

The `exit` command is used to exit the shell.
