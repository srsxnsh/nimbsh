# The `system` module.

The `system` module contains multiple useful commands for getting info about the computer and os.

## `who`

The `who` commands outputs the current user.

```
> who

srsxnsh
```

## `up`

The `up` command outputs the system uptime in seconds.

```
> up

1984.27
```

## `whatos`

The `whatos` command displays operating system info.

```
> whatos

Linux version 6.12.68-1-lts (linux-lts@archlinux) (gcc (GCC) 15.2.1 20260103, GNU ld (GNU Binutils) 2.45.1) #1 SMP PREEMPT_DYNAMIC Fri, 30 Jan 2026 11:42:40 +0000

```
