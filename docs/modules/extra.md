# The `extra` module.

The `extra` module contains various fun and (optionally) useful utilities.

## `spam`

The `spam` command spams the terminal with the input, or otherwise with `"nimbsh!!!"`.

Similar to the UNIX `yes` command.

```
> spam damn!!!

damn!!!
damn!!!
damn!!!
...
```
